### Regex Patterns Needed:
```python
PAN_PATTERN = r"^[A-Z]{5}\d{4}[A-Z]$"
AADHAAR_PATTERN = r"^\d{12}$"
```

### Cross-Field Validations:
1. **DOB → Age:** Calculated age should match stated age (±1 year)
2. **PIN → State:** PIN prefix should match state (e.g., 400xxx = Maharashtra)
3. **Premium → Sum_Assured:** Premium should be < Sum Assured
4. **Gender → Title:** "Mr." ↔ Male, "Mrs./Ms." ↔ Female

## EXTRACTION STRATEGY RECOMMENDATIONS

### Optimal Approach:
1. **Table Detection:** Nemotron-table-structure or fallback to static coordinates
2. **Cell Extraction:** Crop each value cell individually
3. **OCR:** PaddleOCR (EN+HI) for printed/clear text
4. **VLM Fallback:** Qwen-VL only for:
   - Low confidence (< 0.70)
   - Cursive handwriting
   - Critical fields
   - Ambiguous characters

### Special Handling:
- **Checkboxes:** Pixel density analysis, not OCR
- **Dates:** Multiple format support (DD/MM/YYYY, DD-MM-YYYY, DD.MM.YYYY)
- **Names:** Phonetic matching for misspellings
- **Addresses:** Multi-line concatenation
- **PIN Codes:** Prefix lookup for state validation
